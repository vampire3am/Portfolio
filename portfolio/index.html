Interactive Photo Gallery

Project Overview

An interactive photo gallery built with JavaScript that allows users to filter, search, and view images with smooth animations and transitions. The gallery features a responsive grid layout, lightbox functionality, and category filtering.

Key Features

  • Responsive CSS Grid layout that adapts to any screen size
  • Advanced filtering system with multiple category support
  • Custom lightbox with touch support and keyboard navigation
  • Image lazy loading for improved performance
  • Search functionality with real-time results
  • 3D hover effects using CSS transforms

Technologies Used

HTML5 CSS3 JavaScript CSS Grid CSS Grid Intersection Observer API
